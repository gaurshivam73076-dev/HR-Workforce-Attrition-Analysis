# HR Analytics: Workforce Attrition & Retention Insights

## 🎯 Project Overview
This project focuses on analyzing employee turnover trends within an organization using data analytics. The goal is to uncover key insights related to employee demographics, salary brackets, and workload to help HR drive efficiency and improve retention rates.

## 🛠️ Key Tools & Features Used
- *Pivot Tables & Pivot Charts:* Created dynamic visual components for multi-dimensional data breakdown.
- *Power Query:* Automated data cleaning and transformation processes.
- *Data Modeling:* Combined core workforce metrics seamlessly.
- *Slicers:* Added interactive filter panels for Gender, Marital Status, and Business Travel.
- *Advanced Formulas:* Utilized precision retrieval functions for key insights.
